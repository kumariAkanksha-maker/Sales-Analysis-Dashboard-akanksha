# Sales Analysis Dashboard

## Problem Statement
Analyze sales data to understand revenue, profits, and trends across regions and categories.

## Tools Used
- Power BI
- Excel

## Data Cleaning
- Removed blank rows
- Formatted date and numbers
- Renamed columns

## Insights
- West region has highest sales.
- Technology category is most profitable.
- Higher discounts lead to reduced profits.
